# ESP32-drone-keyboard-control
Control ESP32 drone using your computer keyboard.
## Keyboard control method
```
↑:Forward
↓:Back
←:Left
→:Right
A:Counterclockwise
D:Clockwise
W:Lift
S:Drop
Tab:take off
B:stop
SPACE:landing
```
